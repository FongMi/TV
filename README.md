### 點播欄位

<pre><code>searchable：搜索    (0：關閉、1：啟用)  
filterable：篩選    (0：關閉、1：啟用)  
changeable：換源    (0：關閉、1：啟用)
playerType：播放器  (1：IJK、2：EXO)</code></pre>

### 直播欄位

<pre><code>ua：用戶代理
epg：節目網址
logo：台標網址
boot：是否自動開啟
playerType：播放器</code></pre>
